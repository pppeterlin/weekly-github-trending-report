## 三、 Project Detailed Introduction

### 1. colbymchenry/codegraph (排名: 19) **【領域精選】**
- **核心功能**: 在本地對整個專案做預索引，輸出可被 Claude Code 直接查詢的程式碼知識圖譜，減少模型反覆 Grep / Read 的 tool call。
- **推薦原因**: 同時打中你的「數據工程（離線索引 + 圖結構儲存）」與「記憶引擎（Code-level 長期記憶）」兩個重點領域；連週上榜代表社群驗證度高。
- **技術亮點**: 100% 本地化、預索引架構，把語意檢索成本從 inference time 移到 build time——與 Memosyne 思路同源。

### 2. anthropics/claude-code (排名: 9)
- **核心功能**: 官方 agentic 終端編碼工具，用自然語言指令驅動檔案編輯、命令執行與多步任務。
- **推薦原因**: 你日常 Agent 工作流的核心載體，理解其工具呼叫與權限模型對自建 Agent 有直接參考價值。
- **技術亮點**: ReAct 式的 tool-use 迴圈 + 權限分級設計，是 production-grade coding agent 的範本。

### 3. run-llama/liteparse (排名: 25) **【領域精選 / Data·RAG】**
- **核心功能**: LlamaIndex 團隊出品的輕量高速文件解析器，把 PDF / Office / 各式文檔轉成結構化片段。
- **推薦原因**: RAG pipeline 的前處理基石，數據工程視角下是「文件 → 向量庫」鏈路中最易踩坑的一段，值得直接導入。
- **技術亮點**: 主打 fast + open-source，相較重型 parser 在吞吐與部署成本上更友善，適合自建知識庫批次處理。

### 4. D4Vinci/Scrapling (排名: 11) **【領域精選 / Data Engineering】**
- **核心功能**: 自適應網頁爬取框架，當目標頁面結構改變時能自動調整選擇器、維持抓取穩定性。
- **推薦原因**: 純數據工程題材，對需要長期維護資料來源的爬蟲管線特別實用，可大幅降低維運成本。
- **技術亮點**: 「adaptive selector」機制把脆弱的 XPath / CSS 規則做成可自癒，是爬蟲工程化的好範例。

### 5. microsoft/markitdown (排名: 2)
- **核心功能**: 微軟開源工具，將 Office 文件、PDF、圖片等多格式統一轉換為 Markdown。
- **推薦原因**: 是把企業既有文件餵給 LLM / RAG 的標準入口，數據工程與記憶引擎的共同前置層。
- **技術亮點**: 統一輸出 Markdown 作為 LLM-friendly 中介格式，極大簡化多源異構文件的 ingestion 流程。

### 6. revfactory/harness (排名: 16) **【領域精選 / Agent】**
- **核心功能**: 一個 meta-skill，用來設計「領域專屬的 Agent 團隊」，定義角色、分工與協作協議。
- **推薦原因**: 把 Skill 從「單一能力」抽象到「團隊編排」層，正是你規劃多 Agent 工作流時需要的方法論。
- **技術亮點**: 以 skill 定義 agent team 的組織結構，是 multi-agent orchestration 的可重用模板。

### 7. obra/superpowers (排名: 27) **【領域精選 / Agent】**
- **核心功能**: Agentic Skills 框架與配套的軟體開發方法學，提供 Skill 撰寫規範、評測與組合策略。
- **推薦原因**: 與 Anthropic Skills 生態高度兼容，是把個人經驗沉澱為可重用 Skill 的最佳起點。
- **技術亮點**: 完整的 Skill 生命週期（撰寫→測試→版本→分發），方法學層面比個別 Skill Pack 更有長期價值。

### 8. shiyu-coder/Kronos (排名: 24) **【領域精選 / Data·Finance】**
- **核心功能**: 為金融市場「語言」打造的 foundation model，對價格 / 交易序列做表徵與預測。
- **推薦原因**: 數據工程的跨域延伸——把時序金融數據當作 token 序列建模，方法論可借鑑到其他高頻時序資料。
- **技術亮點**: 「market-as-language」的 tokenization 思路，是時序基礎模型設計的新興範式。

### 9. Lum1104/Understand-Anything (排名: 3)
- **核心功能**: 把程式碼或文件轉成可互動的知識圖譜介面，支援節點展開、語意搜尋。
- **推薦原因**: Memory/RAG 視覺化層代表，與 codegraph 互補，前者偏 UX、後者偏底層索引。
- **技術亮點**: 把 graph traversal 暴露為 Agent 可調用的 API，是 RAG 圖結構走向交互式探索的好示範。

### 10. manaflow-ai/cmux (排名: 23) **【領域精選 / Agent Tooling】**
- **核心功能**: 為 AI coding agent 設計的 macOS 終端，用垂直分頁同時管理多個並行 agent session。
- **推薦原因**: 你在 macOS 上跑多個 Claude Code / Agent 時的實用工具，提升並行任務的可視化與切換效率。
- **技術亮點**: 把「多 agent 並行」做成第一公民的終端 UX，是 agent-native 開發環境的好參考。

### 11. affaan-m/ECC (排名: 4)
- **核心功能**: 整合 skills 與安全控制的 Agent 效能優化系統。
- **推薦原因**: 關注 Agent 在 production 中的效能與安全治理時可作為參考實作。
- **技術亮點**: 把 skill 載入與安全邊界放在同一框架，體現 agent governance 的工程思路。

### 12. EveryInc/compound-engineering-plugin (排名: 8)
- **核心功能**: Claude Code 與相容編輯器的官方外掛，主打「複利式工程（compound engineering）」工作流。
- **推薦原因**: 提供把重複工程動作系統化累積的範式，與你沉澱個人 Skill / 流程的目標一致。
- **技術亮點**: 以外掛形式封裝可組合的工程慣例，便於團隊間傳播最佳實踐。

### 13. anthropics/knowledge-work-plugins (排名: 5)
- **核心功能**: Anthropic 為知識工作者打造的一組開源 Claude 外掛。
- **推薦原因**: 官方品質的外掛範例，可作為自寫外掛的格式與規範對照。
- **技術亮點**: 展示 Anthropic 對非程式類知識工作場景的 Agent 化設計取向。

### 14. galilai-group/stable-worldmodel (排名: 29) **【領域精選 / Research·Data】**
- **核心功能**: 提供可重現的 world model 研究平台，含模擬環境、數據與評測基建。
- **推薦原因**: 對模擬數據生成與可重現實驗有興趣時值得關注，是 research infra 的數據工程面。
- **技術亮點**: 強調 reproducibility 的實驗框架設計，對建立可靠的數據/模型評測流程有參考價值。

### 15. multica-ai/andrej-karpathy-skills (排名: 26)
- **核心功能**: 把 Karpathy 對 LLM coding 的觀察整理為可載入的 Claude Skill / CLAUDE.md。
- **推薦原因**: 高訊噪比的 prompt 工程資料，可直接套到你日常 Claude Code 流程。
- **技術亮點**: 將「名家觀察」結構化為 Skill 是新型 knowledge packaging 模式，值得仿作個人 Skill Pack。
