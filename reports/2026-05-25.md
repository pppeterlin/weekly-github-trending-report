## 三、 Project Detailed Introduction

### 1. colbymchenry/codegraph (排名: 1)
- **核心功能**: 在本地對整個專案做預索引，輸出可被 Claude Code 直接查詢的程式碼知識圖譜，減少模型反覆 Grep / Read 的 tool call。
- **推薦原因**: 同時打中你的「數據工程（離線索引 + 圖結構儲存）」與「記憶引擎（Code-level 長期記憶）」兩個重點領域；對 Agent dev workflow 直接可用。
- **技術亮點**: 100% 本地化、預索引架構，跟 Memosyne 思路同源——把語意檢索成本從 inference time 移到 build time。

### 2. rohitg00/agentmemory (排名: 18) **【領域精選】**
- **核心功能**: 為 Coding Agent 設計的持久記憶系統，支援跨 session 的事實/偏好/任務狀態儲存。
- **推薦原因**: 直接對應你關注的「長短期記憶引擎」，且是少數有 benchmark 支撐的 memory layer 實作。連週上榜，社群驗證度高。
- **技術亮點**: 將 memory 拆成 episodic / semantic / procedural 三層，可作為個人 Memosyne 架構的參考實作。

### 3. K-Dense-AI/scientific-agent-skills (排名: 13) **【高優先 / Biotech】**
- **核心功能**: 一組可即插即用的科研 Skills，覆蓋生物、化學、工程、金融四大領域的常見 research workflow。
- **推薦原因**: Biotech 為你的加分項，本項目是本週唯一明確覆蓋生物信息學 Agent workflow 的高排名專案，值得直接 fork 試跑。
- **技術亮點**: 把 wet-lab / dry-lab 常見步驟拆成可組裝的 Skill 單元，是「Domain-specific Agent」設計範式參考。

### 4. HKUDS/CLI-Anything (排名: 6)
- **核心功能**: 提供統一的 CLI 抽象層，讓任意桌面/伺服器軟體都能被 Agent 直接呼叫。
- **推薦原因**: 是 Agent 落地的「最後一哩」工程化基礎，跟 MCP 形成互補；HKUDS 團隊出品，學術品質有保證。
- **技術亮點**: Agent-Native 介面協定設計，可作為內部數據工具 Agent 化的模板。

### 5. obra/superpowers (排名: 11) **【領域精選】**
- **核心功能**: Agentic Skills 框架與配套的軟體開發方法學，提供 Skill 撰寫規範、評測與組合策略。
- **推薦原因**: 與 Anthropic Skills 生態高度兼容，是把個人經驗沉澱為可重用 Skill 的最佳起點。
- **技術亮點**: 完整的 Skill 生命週期（撰寫→測試→版本→分發），方法學層面比個別 Skill Pack 更有長期價值。

### 6. rtk-ai/rtk (排名: 25) **【領域精選 / Data Infra】**
- **核心功能**: 作為 CLI Proxy 攔截常見 dev command，對 LLM 請求做 cache / diff / 壓縮，宣稱可省 60–90% token。
- **推薦原因**: 數據工程視角下的 LLM Infra：以 pipeline 思維處理 token 流，與你日常使用 Claude Code 高度相關，可直接導入降本。
- **技術亮點**: 「LLM 旁路代理」是新興細分賽道，rtk 的 diff/cache 策略可借鑑到自建 Agent 上。

### 7. earendil-works/pi (排名: 19) **【領域精選】**
- **核心功能**: 一體化 Agent toolkit：CLI Coding Agent、Unified LLM API、TUI/Web UI 三件套。
- **推薦原因**: 想自建私有 Agent 棧時，可以直接拿 pi 當骨架，省去整合 LLM provider 與 UI 的繁雜工程。
- **技術亮點**: Unified LLM API 抽象層做得乾淨，TUI 與 Web UI 共用同一 agent core，是不錯的模組化案例。

### 8. Lum1104/Understand-Anything (排名: 3)
- **核心功能**: 把程式碼或文件轉成可互動的知識圖譜介面，支援節點展開、語意搜尋。
- **推薦原因**: Memory/RAG 視覺化層代表，與 codegraph 互補，前者偏 UX，後者偏底層索引。
- **技術亮點**: 把 graph traversal 暴露為 Agent 可調用的 API，是 RAG 圖結構走向交互式探索的好示範。

### 9. ruvnet/RuView (排名: 9)
- **核心功能**: 利用 WiFi CSI 訊號進行空間定位與生命體徵（心率、呼吸）監測。
- **推薦原因**: Biotech 邊界題材：把消費級無線訊號做成醫療感測，是「無接觸生理監測」研究方向的開源實作。
- **技術亮點**: 信號處理 + 機器學習 pipeline 完整，對訊號數據工程有興趣可作為參考。

### 10. tinyhumansai/openhuman (排名: 4)
- **核心功能**: 主打私有部署的個人 AI Agent，整合多模型與工具。
- **推薦原因**: 探討個人 super-intelligence 形態的參考實作，與「個人 Memosyne + Agent」終態接近。
- **技術亮點**: Privacy-first 架構與本地工具整合策略，可借鑑其 sandbox 與授權設計。

### 11. multica-ai/andrej-karpathy-skills (排名: 8)
- **核心功能**: 把 Karpathy 對 LLM coding 的觀察整理為可載入的 Claude Skill。
- **推薦原因**: 高訊噪比的 prompt 工程資料，可直接套到你日常 Claude Code 流程。
- **技術亮點**: 將「名家觀察」結構化為 Skill 是新型 knowledge packaging 模式，值得仿作個人 Skill Pack。

### 12. Imbad0202/academic-research-skills (排名: 7)
- **核心功能**: 學術研究全流程 Skill：題目發想 → 文獻 → 撰寫 → 評審 → 定稿。
- **推薦原因**: 若做 biotech 或 data 相關研究筆記/論文整理，可直接接入。
- **技術亮點**: Skill 之間有明確的階段 contract，可學習其 workflow chaining 設計。

### 13. dotnet/skills (排名: 10)
- **核心功能**: 微軟官方為 AI Coding Agent 出品的 .NET / C# Skill 集。
- **推薦原因**: 雖非你日常技術棧，但作為大廠 Skill 工程化範例值得快速瀏覽。
- **技術亮點**: 官方規範化的 Skill 描述與測試結構，可作為自寫 Skill 的格式對照。

### 14. rohitg00/ai-engineering-from-scratch (排名: 5)
- **核心功能**: 從零打造 AI 工程能力的實作型課程。
- **推薦原因**: 對團隊培訓或 onboarding 新成員 AI 工程素養有幫助，自身可選讀進階章節。
- **技術亮點**: 章節從 prompt → eval → deploy 完整，少數聚焦「shipping」而非僅 prototype 的教材。

### 15. anthropics/claude-plugins-official (排名: 2)
- **核心功能**: Anthropic 維護的官方 Claude Code Plugin 目錄。
- **推薦原因**: 屬資源索引，主要用來追蹤官方 Plugin 動態。
- **技術亮點**: 可從中觀察 Anthropic 對「官方品質」Plugin 的標準與篩選方向。
