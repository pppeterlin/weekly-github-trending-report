## 三、Project Detailed Introduction

### 1. [topoteretes/cognee](https://github.com/topoteretes/cognee)（排名：28）⭐ 領域精選

- **核心功能**：cognee 是面向 AI Agent 的開源知識引擎，結合向量搜索與知識圖譜雙架構，支援多模態數據注入，提供 `remember / recall / forget / improve` 四個核心操作，實現 Agent 的長短期記憶持久化。
- **推薦原因**：直接對應「個人記憶引擎」核心需求——動態知識圖譜構建、跨 session 持久記憶、多租戶隔離設計，符合生物信息 Agent 的上下文長期學習需求，也適合整合至數據工程知識蒸餾流程。
- **技術亮點**：向量 + 圖譜自動路由查詢（auto-routing dual search）；企業級 OTEL 可觀測性；支援 Modal / Railway / Fly.io 自託管；Python 86.5%，已達 v1.0.1 生產穩定版。

---

### 2. [openai/openai-agents-python](https://github.com/openai/openai-agents-python)（排名：9）

- **核心功能**：OpenAI 官方輕量級多 Agent 工作流框架，支援 Agent 委派、工具整合（含 MCP）、安全護欄、人工介入點、對話歷史管理與語音 Agent。
- **推薦原因**：連續 6 天上榜代表社群長期信任；MCP 協議原生支持與 100+ LLM 提供商相容，意味著可與現有 Claude/Anthropic 工具無縫對接；生產級護欄設計適合需要穩定性的 Biotech 數據管線。
- **技術亮點**：內建 Tracing UI 調試儀表板；Sandbox Agent 支援容器化環境中的實際文件系統操作；Agent 組合式架構（agents as composable tools）。

### 3. [obra/superpowers](https://github.com/obra/superpowers)（排名：21）⭐ 領域精選

- **核心功能**：為 AI Coding Agent 設計的七段式軟體開發方法論框架，覆蓋設計 → Git worktree 隔離 → 計劃 → 子 Agent 執行 → TDD → 代碼審查 → 分支合併全流程，技能自動觸發無需手動調用。
- **推薦原因**：與 Claude Code 深度整合，技能模組化設計（測試、調試、協作）可直接加速數據工程腳本開發與 Agent 工作流搭建；本週新增 13,900 顆星顯示社群爆發式採用。
- **技術亮點**：Shell 主導（61.6%）的輕量設計；跨平台支援 Claude Code、Cursor、OpenAI Codex、Gemini CLI；強制 TDD 工作流確保代碼品質。
